# MODAKBUL



> 세종대학교 제 3대 소프트웨어융합대학교 학생회 "번영"에서 주최하는 웹 소설 공모전으로, 소프트웨어융합대학의 기존페이지의 문제점들을 파악하며 새로운 단과대학 페이지로 리뉴얼을 하는 것을 목표로 잡고 있다.
>
> 본 프로젝트는 해당 공모전에서 "대상"을 수여한 작품이다.
>
